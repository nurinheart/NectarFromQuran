COMMON ISSUES & FIXES
==========================================

Issue: Workflow fails with "Instagram login error"
Fix: Re-run get_instagram_session.py and update INSTAGRAM_SESSION_DATA secret

Issue: Posts not appearing at scheduled time
Fix: GitHub Actions can delay up to 10 minutes; workflow may be disabled

Issue: Arabic text not rendering
Fix: Check workflow logs; GitHub Actions installs fonts automatically

Issue: "No module named 'cairocffi'"
Fix: Normal on Mac; GitHub Actions installs Cairo automatically

Issue: Wrong tafsir content
Fix: Should say "Tazkirul Quran" not "Ibn Kathir"; check auto_tafsir_fetcher.py
